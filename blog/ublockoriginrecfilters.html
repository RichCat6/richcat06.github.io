Recommended Filter Lists For uBlock Origin

written by RichCat06 on February 18, 2023

In this blog article, we will be talking about a certain feature of uBlock Origin that makes it such a kickass tool: FILTER LISTS!! And I will be listing off some filter lists that I recommend to make uBlock Origin stronger. If you see some links at the bottom of the list entries saying "Copy this into uBlock," they are just there if you would like to add them to your uBlock Origin stuff.

FuckFuckAdBlock

This filter list blocks ad block detection on certain websites (for example, The New York Times.) It does a pretty good job of it too! Copy this into uBlock!

OISD

This one is a compilation of several other lists and can help get past more sneaky websites. Copy this into uBlock!
