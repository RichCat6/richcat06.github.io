Software I Use

Below is a list of software that I use to make this website. Just about all the software I use is open source and doesn't track you or sell your data to third parties.

IMAGES

To make some of the images that I use on this website, I use the GNU Image Manipulation Program or GIMP. It's a good open source alternative to Photoshop, which tracks you and sells your data. When you use Photoshop, you don't own the software (they have this rentware model where you have to pay 10 bucks a month to use it. They're nuts if they think I'd pay a monthly fee for something like that!)

WEB PAGES

To make the web pages for this website, I use Bluefish, which, in my opinion, is MILES better than Visual Studio Code. It has a feature where you can add tags to your HTML without even having to go through the hassle of typing them, which makes editing these pages ten times easier, and a hell of a lot faster!
